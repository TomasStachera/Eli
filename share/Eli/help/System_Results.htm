Command System_Results

Using of command

This command is use for display results in result window.User can select which variable will be display in result window and name of this variable.It can be select which picture will be display in picture result window.

Commands parameters

Command has 5 parameters:
  1. Displayed picture: It can be select if some picture will be displayed in picture window.It can be select option "None", no picture will be display. Next option is Picture 0-19. Selected picture will be display in picture window.
  2. Result name: Name of first result. This name will be displayed before first result. Selected variable in the next parameter will have "name"
  3. Result variable: Variable which will be display after Result name sets in previous parameter. You can select option "None", than any variable will be display.Next options are Variable_0 to Variable_99. The last option is String_0 to String_9. Value of selected variable or string will be display.
  4. Next 2 parameters are same as previous 2 parameters,it inserts next line with result
  5. Result name: Name of object result. Name will be displayed before object result
  6. Object parameters: It can be selected which object parameters will be displayed in result window
  7. Object name: Name of object which parameter will be displayed
  8. Unit: It can be selected if object parameter will be displayed in pixels or real world units (option mm). It can not be only milimeters, it can be meters, inch or others units. Size coeficient must be setup for this unit in Source_ReadVideo, Source_ReadCamera or Source_OpenFile command.
  9. Undistortion chan: It must be select which channel will be used for real world unit calculation

Error codes

  1. Error code -11:Bad picture name.Name of displayed picture has not correct format
  2. Error code -12:Bad variable 1 name.Variable 1 name has not correct format
  3. Error code -13:BAd variable 2 name. Variable 2 name has not correct format
  4. Error code -14: Bad undistortion channel. Check if undistortion channel was set
  5. Error code -15:Error get object value. It was error durring get object parameter. Check if object with given name exist
