Object_ToChar

Using of command

Function sets ASCII character to specefied objetc. Function Object_Find must be called before this function.

Commands parameters

Command has 2 parameters:
  1. Object name: You need write object name, which object will be set to character
  2. Character: ASCII character which will be set to object specified in previous line.

Error codes

  1. Error code -11:No character found.No character was set or space character was set.
