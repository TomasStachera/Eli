Object_Find command page

Object Find edit window

This window is displayed in debug mode when Run One step command is executed in command Object_Find



Window has 6 areas:
  1. Menu window
  2. Picture window
  3. Contour window
  4. Buttons
  5. Detail window
  6. Coordinates window

    7. Menu window

    Menu window has several buttons which for working with selected picture
    1. Close This button closes Object window and return to main window to the line with Object_Find command. Changes are not saved automatically, so press Save button before click on Close button.

    2. Save>This button save all changes done on this window to eli program.Changes are saved only in eli program not in eli file, so you need to save eli program in main application window as those saved changes was saved also on .eli file.

    3. Show Contours Click on this button draw all selected contours from Contour window to Picture window

    4. Filter moduleFilter modole window will display after click on this button. You can program filter for contour. Filter module window is described here: Filter module window

    5. Run FilterClick on this button executes filter module program.Only conours which passed filter are displayed in the Contour window and only passed contours are used in the object calculation.

    6. Clear filterFunction cancel filter execution. It means that all contours are displayed in the Contour window and all contours can be used in the object calculation.

    7. Object moduleClick on this button opens Object modulle window where you can done all objects operations (Create objects, copy, delete objects, debug objects).Object module window is describe here: Object module window

    8. Find objectsFunction executes Find object module.Function find all objects programmed in the Object module window. Result of this operation is displayed in the Detail window. Number of found each type of objects and contours describing the object.

    9. Object List.Function display object list window if some objects was found in the Find objects function.(Find object must be executed before click on this button). If no objects was found error message will display after click on this command and Object list window will not show. Object list window has list box with all found objects. left click on selected object will display basic information about object (Bouding rectange: x,y,width,height, perimeter and contour area)Click on button Show object close this window and display selected object on the Picture window. Click on button Exit just close this window.

    10. Clear Objects. Click on this button clear Picture window fromall displayed objects or contours. Only source image will stay on this window.

    11. SetupClick on this button open Setup window.This window setups displayed object parameters in the Picture window. It can be setup Object line parameters (color, thickness). Object line is displayed area in the Picture window around selected object in Object list window. This parameter is also used in the Show contours option. Next part of this window is object name.It can be selected object name parameters (color, line thickness, line type and letter font).Object name is name of the object display in the Picture window ower selected object in Object list window. After change some parameter you need to click Update button for save changes. Click on button Exit close this window.

    Picture window

    Picture window display source image which is selected in the function Object_Find. This picture must be binary image.Picture window display also contours or objects after click on proper menu button. When you move mouse on the Picture window area mouse coordinates and contours over the cursor will be display in the Coordinates window.

    Contour window

    Contour window display all found contours on the source image or all passed filter contours if Run filter option was executed. Left click on any contour will display all contours parameters in the Detail window on selected contour. Left click can select or unselect one contour. Only selected contour will be display in picture window when Show contour option will be selected.

    Buttons

    There are 3 buttons on this area:
    1. All: Click on this button selects all contours in the Contour window
    2. None: Click on this button unselect all contours in the Contour window
    3. Export: Click on this button exports all selected contours in the Contour window to the file. After click on this button export window will display. It can be select in the selected window which group of parameters will be exported to the file.When you select option Display file, exported file will be displayed. Click on button Export save file dialog will open. You need to select directory and write export file name with extension.According file extension , program execute software for export file display (If you choose option Display file).Each contour is on the separate line. Contours parameters are separete with horizontal tabelator (\t). Click on button Cancel just close this window.

    Detail window

    This window display all contours parameters when you left click on some contour in the Contour window Explanation of contour parameters is describe here: Contour parameters . When Find object function will be executed this window display information about number of found object and which contour is which object.

    Coordinates window

    This window display coordinates of cursor position when you move in to Picture window. It display also Contour number outside cursor position (also contours which not pass after filter execution). Remember that there is always one contour over all picture area so it is always display on this window ( You can filter it in the filter) but it will be stil display on this window.
